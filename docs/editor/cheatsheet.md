> On iPad, you can also long press the `⌘` key to show all keyboard shortcuts.

**Navigation**

| Action                  | Key      |
| :---------------------- | -------: |
| Show/Hide sidebar       | ⌘ E      |
| Open Files              | ⌘ O      |
| Close File              | ⌘ W      |
| New File                | ⌘ N      |
| New Tab                 | ⌘ T      |
| Next Tab                | ⌘ ⌥ →    |
| Previous Tab            | ⌘ ⌥ ←    |
| Reveal in Explorer      | ⌘ ⇧ J    |
| Find in Document        | ⌘ F      |
| Show Outline            | ⌘ ⌥ O    |
| Toggle Preview          | ⌘ P      |
| Run Actions             | ⌘ R      |

**Font Size**

| Action                  | Key      |
| :---------------------- | -------: |
| Increase Font Size      | ⌘ +      |
| Decrease Font Size      | ⌘ -      |
| Reset Font Size         | ⌘ 0      |

**Text Formatting**

| Action                  | Key      |
| :---------------------- | -------: |
| Toggle Bold             | ⌘ B      |
| Toggle Italic           | ⌘ I      |
| Toggle Underline        | ⌘ U      |
| Toggle Strikethrough    | ⌘ D      |
| Toggle Heading Level    | ⌘ 1 ~ 6  |

**Text Editing**

| Action                  | Key      |
| :---------------------- | -------: |
| Move Lines Up           | ⌘ ⌥ ↑    |
| Move Lines Down         | ⌘ ⌥ ↓    |
| Left Indent             | ⌘ [      |
| Right Indent            | ⌘ ]      |

**Quick Actions**

| Action                  | Key      |
| :---------------------- | -------: |
| Insert Link             | ⌘ K      |
| Open Link/File          | ⌘ ⇧ O    |
| Insert Code Block       | ⌘ ⇧ C    |
