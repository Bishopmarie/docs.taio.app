**页面导航**

| 动作 | 快捷键 |
| :---- | ----: |
| 展开/收起侧边栏 | ⌘ E |
| 打开文件 | ⌘ O |
| 关闭文件 | ⌘ W |
| 新建文件 | ⌘ N |
| 新建标签 | ⌘ T |
| 切换到下个标签 | ⌘ ⌥ → |
| 切换到上个标签 | ⌘ ⌥ ← |
| 在文档中查找 | ⌘ F |
| 显示大纲 | ⌘ ⌥ O |
| 切换预览 | ⌘ P |
| 运行动作 | ⌘ R |

**文本字号**

| 动作 | 快捷键 |
| :---- | ----: |
| 加大字号 | ⌘ + |
| 减小字号 | ⌘ - |
| 还原字号 | ⌘ 0 |

**文本格式**

| 动作 | 快捷键 |
| :---- | ----: |
| 切换粗体 | ⌘ B |
| 切换斜体 | ⌘ I |
| 切换下划线 | ⌘ U |
| 切换删除线 | ⌘ D |
| 切换标题等级 | ⌘ 1 ~ 6 |

**文本编辑**

| 动作 | 快捷键 |
| :---- | ----: |
| 向上移动行 | ⌘ ⌥ ↑ |
| 向下移动行 | ⌘ ⌥ ↓ |
| 减少缩进 | ⌘ [ |
| 增加缩进 | ⌘ ] |

**快捷动作**

| 动作 | 快捷键 |
| :---- | ----: |
| 插入链接 | ⌘ K |
| 打开链接或文件 | ⌘ ⇧ O |
| 插入代码块 | ⌘ ⇧ C |
